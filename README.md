# LcdGame
Проект на Arduino 
- Гонки на LCD экране (I2C)
- Две версии (V1 и V2)

# Подключение к Arduino:
Lcd дисплей:

SDA - A4

SCL - A5

VCC - 5V

GND - GND

Джойстик:

VCC - 5V
 
GND - GND
 
X - A0
 
Y - A1
 
B - 2 (digital)
